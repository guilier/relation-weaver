## 插件简介

### 这是什么？

「人物关系谱系」是一款专为**创作者**设计的 Obsidian 插件，特别适合：

- ✍️ **小说作者** - 管理小说人物关系
- 🎮 **跑团主持人** - 记录 NPC 关系和剧情时间线
- 📚 **世界构建者** - 梳理世界观中的人物网络
- 📝 **读书笔记** - 分析书中人物关系

### 它能做什么？

| 功能 | 说明 |
|------|------|
| 👥 **人物管理** | 从 Markdown 文件自动解析人物信息 |
| 🔗 **关系管理** | 手动或自动建立人物间的关系网 |
| 🏰 **阵营管理** | 按阵营/势力分组查看人物 |
| 📅 **时间线** | 记录事件，自动关联人物出场 |
| ⏳ **生命周期** | 可视化人物出生→出场→死亡时间轴 |
| 📊 **数据统计** | 自动生成各种统计图表 |
| 🏷️ **标签系统** | 自定义事件标签，灵活筛选 |
| 💾 **导入导出** | 支持 JSON/CSV 数据迁移 |

## 快速上手

### 1️⃣ 安装插件

**方式一：手动安装**

1. 下载插件文件到 `.obsidian/plugins/my-char-view/`
2. 在 Obsidian 设置 → 社区插件 → 启用「人物关系谱系」

**方式二：通过 BRAT 安装**（如果插件已发布）

### 2️⃣ 创建数据文件

在仓库根目录创建两个文件：

**📄 人物索引.md**

```markdown
## 张三
- 身份：皇帝
- 阵营：北境王国
- 首次出场：公元前300年
- 出生：公元前280年
- 死亡：前200年
- 亲密人物：李四

## 李四
- 身份：将军
- 阵营：北境王国
- 出生：290年
- 死亡：350年

## 王五
- 身份：部落首领
- 阵营：南疆部落
- 首次出场：310年
```

**📄 时间线.md**

```markdown
## 300年
### 春季：
- [战争] 张三率军出征
- [政治] 李四受封将军

### 秋季：
- [感情] 张三与某公主成婚

## 310年
### 夏季：
- [出场] 王五首次登场
```

### 3️⃣ 打开视图

点击左侧边栏的 👥 图标，或通过命令面板执行「打开人物关系视图」

### 4️⃣ 开始使用

点击「🔄 刷新数据」加载你的文件，然后就可以浏览和管理人物关系了！

## 数据文件格式

### 📄 人物索引文件

**文件名**：`人物索引.md`（可自定义）

**格式规则**：

```markdown
## 人物名称          ← 二级标题表示人物
- 字段名：字段值    ← 列表项表示属性
- 阵营：北境王国
- 身份：皇帝
```

**支持的字段**（可在设置中自定义名称）：

| 字段 | 说明 | 示例 |
|------|------|------|
| 阵营 | 人物所属势力 | 北境王国 |
| 身份 | 人物身份/职位 | 皇帝 |
| 出生 | 出生时间 | 公元前280年 |
| 死亡 | 死亡时间 | 前200年 |
| 首次出场 | 首次出现时间 | 公元前300年 |
| 亲密人物 | 亲密关系人物 | 李四 |

**时间格式支持**：

```
✅ 公元前280年
✅ 前100年
✅ 280年
✅ 公元100年
✅ 280         (自动识别为公元)
✅ BC 280
✅ AD 100
```

### 📄 时间线文件

**文件名**：`时间线.md`（可自定义）

**格式规则**：

```markdown
## 300年              ← 二级标题表示年份
### 春季：            ← 三级标题表示月份/季节
- [战争] 张三率军出征  ← 列表项表示事件，[标签] 可选
- [政治] 李四受封将军

### 秋季：
- [感情] 张三与公主成婚
```

**标签说明**：
- 使用 `[标签名]` 格式标记事件
- 标签可在「标签管理」中自定义
- 未标记标签的事件归类为「其他」

## 功能详解

### 👥 人物视图

**功能**：浏览所有人物，按阵营分组

**交互**：
- 🔍 搜索：按姓名、身份、阵营等搜索
- 📋 点击人物卡片 → 查看人物详情
- 🏷️ 显示出场次数、关系数量

**人物详情弹窗**：
- 显示所有字段信息
- 显示生命周期（出生→首次出场→死亡）
- 显示手动建立的关系
- 显示共同出场人物
- 显示相关时间线事件

### 🏰 阵营视图

**功能**：管理人物阵营

**操作**：
- ➕ 添加阵营
- ✏️ 编辑阵营（名称、颜色、描述）
- 🗑️ 删除阵营
- ▶️ 展开/折叠查看阵营成员

### 🔗 关系视图

**功能**：管理人物间的关系

**操作**：
- ➕ 添加关系（选择人物A、人物B、关系类型、亲密度）
- ✏️ 编辑关系
- 🗑️ 删除关系

**关系属性**：

| 属性 | 说明 |
|------|------|
| 类型 | 父子、夫妻、朋友、敌人等 |
| 亲密度 | -3 到 5 的等级（可自定义） |
| 开始时间 | 关系建立的起始时间 |
| 结束时间 | 关系结束的时间 |
| 描述 | 关系说明 |

**视图模式**：
- 📋 列表视图：按列表显示所有关系
- 📁 分组视图：按关系类型分组显示

**批量操作**：
- 按类型批量删除
- 按类型批量更新亲密度
- 清空所有关系
- 导出关系为 CSV

**智能建议**：
- 自动检测共同出场的人物
- 推荐可能的关系

### 📅 时间线视图

**功能**：查看和管理事件时间线

**标签管理**：
- ✏️ 自由添加/编辑/删除标签
- 🎨 自定义标签颜色
- 🏷️ 按标签筛选事件

**交互**：
- 点击人物名 → 跳转人物详情
- 按标签筛选事件
- 事件按年份/月份分组显示

### ⏳ 生命周期视图

**功能**：可视化人物时间轴

**显示内容**：
- 🟢 出生时间
- 🔵 首次出场时间
- 🔴 死亡时间
- 活跃区间（条形图）

**排序方式**：
- 按出生时间排序
- 按首次出场排序
- 按死亡时间排序

**筛选**：
- 显示全部人物
- 仅显示有时间数据的人物

### 📊 统计视图

**功能**：自动生成数据统计

**统计内容**：
- 📊 数据概览（人物数、事件数、关系数、阵营数）
- ❤️ 亲密度分布
- 🏷️ 事件标签分布
- 🏆 出场排行榜
- 📈 年份事件趋势

**导出**：
- 📄 导出统计报告（Markdown 格式）

### 💾 导入导出视图

**功能**：数据迁移和备份

**导出**：
- 📄 导出为 JSON（完整数据）
- 📊 导出关系为 CSV

**导入**：
- 📂 导入 JSON 文件

## 自定义配置

### ⚙️ 设置面板

打开设置 → 人物关系谱系

#### 📄 人物文件配置

| 设置项 | 说明 | 示例 |
|--------|------|------|
| 人物索引文件夹路径 | 存放人物文件的文件夹 | `人物资料` 或 `Worldbuilding/Characters` |
| 人物索引文件名 | 人物文件的名称 | `人物索引.md` |

#### 📅 时间线文件配置

| 设置项 | 说明 | 示例 |
|--------|------|------|
| 时间线文件夹路径 | 存放时间线文件的文件夹 | `时间线` 或 `Worldbuilding/Timeline` |
| 时间线文件名 | 时间线文件的名称 | `时间线.md` |

#### 🏷️ 事件标签配置

点击「🏷️ 管理标签」打开可视化标签编辑器：

- ➕ 添加标签
- ✏️ 编辑标签名、值、颜色
- 🗑️ 删除标签
- ↺ 重置为默认标签

#### 📌 关系类型配置

点击「📌 管理关系类型」自定义关系类型列表。

**默认类型**：
```
父子, 母子, 父女, 母女, 兄弟, 姐妹, 夫妻, 恋人, 
朋友, 挚友, 战友, 师生, 师徒, 君臣, 主仆, 
敌人, 对手, 仇人, 同事, 同盟, 同期出场, 其他
```

#### ❤️ 亲密度等级配置

点击「❤️ 管理亲密度等级」自定义亲密度等级。

**默认等级**：

| 数值 | 标签 | 颜色 |
|------|------|------|
| -3 | 仇恨 | #c0392b |
| -2 | 厌恶 | #e67e22 |
| -1 | 冷淡 | #f39c12 |
| 0 | 陌生 | #95a5a6 |
| 1 | 认识 | #4a90e2 |
| 2 | 一般 | #2ecc71 |
| 3 | 友好 | #27ae60 |
| 4 | 亲密 | #e74c3c |
| 5 | 挚友/至亲 | #9b59b6 |

#### ⚙️ 其他配置

| 设置项 | 说明 |
|--------|------|
| 人物字段显示顺序 | 控制详情页字段的显示顺序 |
| 阵营字段名 | 标识阵营的字段名 |
| 死亡字段名 | 标识死亡的字段名（支持多个） |
| 出生字段名 | 标识出生的字段名（支持多个） |
| 首次出场字段名 | 标识首次出场的字段名 |
| 亲密人物字段名 | 标识亲密人物的字段名 |

#### 🎨 配置预设

快速切换预设模板：

- **默认（中国古风）**：阵营、身份、出生/死亡等
- **西幻题材**：势力、羁绊、陨落等
- **现代都市**：所属组织、亲密关系等
- **科幻题材**：所属势力、情感链接等

### 🔧 常见工作流

**小说创作工作流**：

1. 在人物文件中创建所有角色
2. 在时间线中记录关键事件
3. 建立角色间的关系
4. 使用统计视图追踪角色活跃度
5. 查看生命周期把握角色时间线

**跑团准备工作流**：

1. 创建 NPC 列表
2. 建立 NPC 间关系
3. 记录重要事件时间线
4. 使用阵营视图管理不同势力
5. 快速查看 NPC 详情
